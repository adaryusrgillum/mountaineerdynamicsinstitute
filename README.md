# Mountaineer Dynamics Institute

Elite performance. Tactical precision. Unapologetic excellence.

## Overview

Mountaineer Dynamics Institute is a high-octane, action-themed website featuring a sleek dark mode design with military-grade aesthetics. Built with pure HTML, CSS, and JavaScript.

## Features

- **Dark Mode Design**: Deep charcoal black (#0a0a0a) background with carbon fiber texture overlay
- **Intense Visual Theme**: Cyan accent colors, dramatic typography, and smooth animations
- **Responsive Layout**: Fully optimized for mobile and desktop devices
- **Interactive Elements**: Smooth scroll navigation, parallax effects, counter animations, and glitch effects
- **Professional Styling**: Military/tactical-inspired language and design

## Getting Started

Simply open `index.html` in your browser, or serve with any HTTP server:

```bash
python3 -m http.server 8000
```

Then navigate to `http://localhost:8000`

## Technologies

- HTML5
- CSS3 (with animations and responsive design)
- Vanilla JavaScript (no frameworks required)
